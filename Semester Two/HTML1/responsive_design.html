What is responsive design? -making your web page adjust its size according to the device's screen size (mobile phone, IPad, Desktop/Macbook) What things will change in the website design between the mobile version and the laptop version? -layouts: column grid layout changes to single column in mobile (mobile - one column grid layout) -text size: size is larger for desktop (laptop) -top navigation: the way the header looks for navigation at the top What is mobile first design? -what you need to do -design the website to look best on mobile FIRST, then make adjusts to make it look good on iPad and desktop
